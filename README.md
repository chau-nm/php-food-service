# php-food-service

## SUMMARY
A basic project about food order is built by php & docker. </br>
The most of target for pratice php skill.

## INSTALL

### Requirement

<table>
  <tr>
    <td>Docker</td>
  </tr>
</table>

### Run command

1. Docker up
```
docker compose up -d
```

2. Docker exec
```
docker compose exec php bash
```

3. Composer install
```
composer install
```

4. Load database
```
composer load-db
```
